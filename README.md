
## Description

KrakenFlex Back End Test Case - Tech Stack: NodeJS (with NestJS) + TypeScript + MongoDB

! Insert your API key and MongoDB connection string to .env file

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
